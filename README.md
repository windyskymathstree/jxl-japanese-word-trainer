# “和风语伴”日语多模态单词学习平台

> 一款集单词学习、听力训练、跟读模仿、课堂互动于一体的多功能日语学习工具。

## 🌟 功能特色

- **多格式词库导入**：支持 `.xls` / `.xlsx` / `.csv` / `.txt` 文件，自动解析第一列（日文）和第二列（中文）。
- **五大学习模式**：
  - 日文学习：随机显示日文单词，点击卡片或音频按钮可听读音。
  - 中文学习：随机显示中文释义，考验日文记忆。
  - 挖空练习：智能隐藏单词中的部分字符，填充式回忆。
  - 听力练习：播放单词读音，用户猜测意思。
  - 单词跟读：专业级复读机功能，可设置朗读次数、间隔、日文/中文语速、自动切换单词，支持日文假名注音。
- **课堂游戏系统**：
  - 学生抽选：导入学生名单，随机滚动抽取。
  - 单词抽选：从词库中随机抽选单词并显示例句。
- **语音合成支持**：使用 Web Speech API，可选择日文/中文不同音色，调节语速。
- **本地数据持久化**：单词列表、学习进度、语音设置等全部保存在浏览器 localStorage 中，关闭页面不丢失。
- **单词管理面板**：可视化编辑单词（日文・中文・词性），支持批量导入/导出，自动修复词性错乱。

## 📥 快速开始

### 方法一：直接运行（无需安装）
1. 下载 `index.html` 文件（或整个项目文件夹）。
2. 使用现代浏览器（Chrome / Edge / Firefox）打开该文件。
3. 点击左侧“选择文件”按钮，导入您的单词表（示例格式见下文）。
4. 点击“导入单词”后，即可在右侧各学习模式中开始使用。

### 方法二：本地服务器运行（可选）
若遇到语音合成或文件读取问题，可启动一个简单的 HTTP 服务：
```bash
# 使用 Python 3
python -m http.server 8080
# 然后访问 http://localhost:8080
```

📁 单词表格式要求
支持三种格式，必须包含日文和中文两列：

格式	示例	说明
Excel (.xls/.xlsx)	第一列：本(ほん) 第二列：书	支持词性第三列（可选）
CSV (UTF-8)	本(ほん),书,名词	逗号分隔，三列为：日文,中文,词性
TXT	本(ほん) 书	制表符或两个空格分隔
💡 日文可以用 单词(假名) 的格式，跟读模式会自动显示假名注音。

🎮 课堂游戏使用说明
学生抽选：准备一个 CSV 文件（无表头），每行一个学生姓名。

单词抽选：准备 CSV 文件（无表头），列顺序为：单词,中文释义,词性,例句（例句可选）。

点击对应上传按钮导入，然后点击“抽选学生”或“抽选单词”即可开始滚动抽选。

⚙️ 高级设置（跟读模块）
朗读次数：1-3 次或自定义。

朗读间隔：单词内重复朗读的间隔时间。

自动切换：朗读完当前单词后，自动跳至下一个单词。

显示选项：可分别控制日文、中文、词性、假名的显示。

音色选择：支持选择系统可用的日文/中文语音包。

随机模式：随机抽取单词，打破固定顺序。

📝 注意事项
由于浏览器的安全策略，语音合成在页面未与用户交互前可能无法自动播放。请点击任意按钮后再使用朗读功能。

某些移动端浏览器对语音合成的支持有限，建议在桌面端 Chrome 上获得最佳体验。

所有单词数据仅存储在您的浏览器本地，不会上传到任何服务器。
